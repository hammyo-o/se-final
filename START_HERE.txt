WHY THE FIX WAS NEEDED:

This only runs during "mvn test" but doesn't create the site/jacoco directory.

Changed to:

This ensures the report is generated during "mvn verify" and creates the
full directory structure: target/site/jacoco/jacoco.xml

This matches the working configuration in testing-agent-demo.
